# Spades Table Prototype

Phase 0 scaffold for the future 1v1 Spades browser prototype.

This app intentionally contains no gameplay UI, card rendering, networking, server, animations, or legacy Spades lobby code.

Future implementation should port architecture concepts from the Euchre prototype:

- identity
- reconnect
- accounts
- leaderboard
- quick match
- tournament history
- clear active room
- free-play wording

The Spades rules engine must come from `packages/spades-core`.
