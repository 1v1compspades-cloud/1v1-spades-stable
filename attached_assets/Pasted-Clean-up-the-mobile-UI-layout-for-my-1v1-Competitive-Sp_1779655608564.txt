The “Last Trick Played” box is way too big and is taking up too much vertical space.

Requirements:

2. Make the box much smaller.

8. It should sit to the right of the center played-card slots, against the right wall.
9. It should NOT push the whole layout down.
10. It should NOT cover playable cards in the player’s hand.

Suggested HTML structure:

Suggested render:

function renderLastCompletedTrick() {
  const box = document.getElementById("lastCompletedTrickDisplay");
  if (!box) return;

  box.innerHTML = `
    <div class="last-trick-title">Last</div>
    <div class="last-trick-card">S1: ${formatCard(game.lastCompletedTrick?.p1, true)}</div>
    <div class="last-trick-card">S2: ${formatCard(game.lastCompletedTrick?.p2, true)}</div>
  `;
}

If my existing formatCard only takes one argument, keep it simple:

function formatCard(card) {
  if (!card) return "—";
  if (typeof card === "string") return card;
  return `${card.value}${card.suit}`;
}
