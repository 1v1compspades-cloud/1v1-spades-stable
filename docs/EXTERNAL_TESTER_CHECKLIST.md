# Spades External Tester Checklist

This beta is free play only. There are no cash prizes, no gambling, no payments, and no tournament payouts.

## Before Testing

- Confirm the visible beta build label at the top of the app.
- Enter a display name.
- Use Real Local Server mode for hosted multiplayer.
- Check the connection status panel before starting.
- If disconnected, use Restore Active Room before taking another action.

## Room Entry

- Create a room.
- Copy/share the room code with another tester.
- Join room by code from a second browser or device.
- If a failed join happens, check the room code and try Join Room again.
- If a room full message appears, expect spectator mode and no player actions.

## Gameplay Flow

- Ready both players.
- Bid from both seats.
- Play at least one hand.
- Confirm current trick, last trick, score, bids, and bags update.
- Confirm stale action messages are understandable if you tap out of turn.

## Reconnect And Disconnect

- Refresh during the room.
- Tap Restore Active Room.
- Confirm the same seat returns.
- Confirm hidden-hand safety still passes.
- Note the AFK/disconnect warning placeholder if you step away.

## Leaving

- Use Leave Room / Back to Lobby when done with a room.
- Rejoin with the room code if continuing.
- Use Back to Lobby / Clear Active Room if the local session gets stuck.

## Reporting Bugs

- Tap the always-visible Report Bug button.
- Fill in issue summary, steps, expected result, and actual result.
- Tap Refresh Diagnostics.
- Tap Copy Diagnostics.
- Send the copied diagnostics plus any screenshots.

Include:
- room code
- phase
- transport mode
- seat
- last action
- last error
- hidden-hand safety status
- connection status
- whether the issue involved create room, join room, Quick Match, bid, play hand, reconnect, or leave room
